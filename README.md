## 📖 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma aplicação simples capaz de **identificar a bandeira de um cartão de crédito** (como Visa, MasterCard, American Express, entre outras) a partir do **número do cartão informado**.

Durante o desenvolvimento foi utilizado o **GitHub Copilot como assistente de codificação**, explorando como **IA generativa pode auxiliar na produtividade do desenvolvedor**, sugerindo trechos de código e auxiliando na construção da lógica da aplicação.

O projeto foi desenvolvido como parte de um desafio prático da **Digital Innovation One (DIO)**.

---

## 🎯 Objetivos do Projeto

* Implementar lógica para **identificação de bandeiras de cartão**
* Explorar o **GitHub Copilot como ferramenta de apoio ao desenvolvimento**
* Praticar **engenharia de prompt aplicada à programação**
* Aplicar **boas práticas de programação em C#**

---

## 🧠 Bandeiras Suportadas

O sistema identifica padrões associados a diversas bandeiras de cartão, como:

* Visa
* MasterCard
* American Express (Amex)
* Discover
* Diners Club
* JCB
* Elo
* Hipercard

Cada bandeira possui **prefixos e formatos específicos**, que são reconhecidos através de validações numéricas.

## 🚀 Como Executar o Projeto

Clone o repositório:

```bash
git clone https://github.com/devrafamon/DIOCopilot.git
```

Acesse a pasta do projeto:

```bash
cd DIOCopilot
```

Execute o projeto utilizando o .NET CLI:

```bash

```
